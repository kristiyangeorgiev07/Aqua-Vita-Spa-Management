# Aqua Vita — СПА & Уелнес Система за Управление

Десктоп приложение (fullscreen) за управление на СПА и уелнес центрове.
Изградено с Python 3, Tkinter и MySQL (XAMPP).

---

## Изисквания

- **Python 3.8+**
- **XAMPP** (Apache + MySQL)
- Python пакети: `mysql-connector-python`, `bcrypt`

---

## Инсталация и настройка

### Стъпка 1 — Инсталирайте зависимостите
```
pip install mysql-connector-python bcrypt
```
или стартирайте `install.bat` (Windows)

### Стъпка 2 — Стартирайте XAMPP
1. Отворете XAMPP Control Panel
2. Стартирайте Apache и MySQL

### Стъпка 3 — Създайте базата данни
1. Отворете: http://localhost/phpmyadmin
2. Import → Изберете файла `database.sql` → Go

### Стъпка 4 — Стартирайте програмата
```
python main.py
```

## Структура на файловете

```
spa_system/
├── main.py                  # Главен файл — стартирайте от тук
├── database.sql             # SQL скрипт — изпълнете веднъж в phpMyAdmin
├── db_config.py             # Конфигурация и автентикация
├── db_settings.py           # Диалог за настройки на DB връзката
├── dal.py                   # Data Access Layer (всички SQL заявки)
├── theme.py                 # Цветова схема и шрифтове
├── widgets.py               # Преизползваеми UI компоненти
├── login_screen.py          # Екран за вход
├── dashboard.py             # Табло с обобщена статистика
├── reception_panel.py       # Регистрация на клиенти
├── visits_panel.py          # Регистрация на посещения (по ID на абонамент)
├── subscriptions_panel.py   # Регистър на абонаменти с търсене
├── admin_panel.py           # Администрация (планове, потребители, справки)
├── reports_panel.py         # Справки и CSV експорт
├── receipt.py               # Генератор на HTML квитанции
├── requirements.txt         # Python зависимости
└── install.bat              # Windows инсталационен скрипт
```

|-----------------|----------|

## Конфигурация на DB

Редактирайте `db_config.py` или използвайте "База данни" в менюто:

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': '',      # XAMPP по подразбиране: празна парола
    'database': 'spa_system',
}
```

Настройките могат да се запазят в `settings.ini` без редактиране на код.
